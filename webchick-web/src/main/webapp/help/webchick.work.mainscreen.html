Main Screen

Main screen provide access to each controller data , such as common data , relays, system state , alarm list . Other controller data are only available through controller data page that can be load by clicking on house name .

When you just opened main screen it will take about 10 second to appear data of first controller on the page . Each data controller sequentially appear on the screen. If for some reason it was not possible to obtain data from the controller in cell value will be -1 .

Figure 1.3. Main Screen start loading data

There are several function provided by the user interface :

  • Data in text box can be changed by user in appropriate format.

  • Option to see list of alarm list by clicking on

  • Last update time is the time the data was changed on screen

  • If there is alarm in the house icon will appear on above the house name

  • Relays are displayed in icons. For example if Fan 1 is On you will see icon
